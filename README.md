This project implements a simple **real-time face Recognition system** using **Python and OpenCV**.  
It captures video from a webcam and draws a **green bounding box** around detected faces.

> Project goal: Learn computer vision basics, webcam handling, and real-time face detection.

---

## Features
- Live video capture from webcam
- Real-time face detection
- Green bounding box around detected faces
- Lightweight and fast
- Exit the application by pressing `q`

---

## Technologies Used
- Python
- OpenCV (cv2)
- Haar Cascade Classifier
